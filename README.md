## 📖 About

**Herbora** is a beautifully crafted, mobile-first web application for exploring medicinal herbs, natural remedies, and holistic wellness knowledge. Built as a single HTML file with no frameworks or build steps — just open and use.

Available in **Romanian 🇷🇴** and **English 🇬🇧**, Herbora brings centuries of plant-based healing wisdom into a clean, fast, and privacy-respecting experience.

> ⚠️ **Disclaimer:** Herbora is for educational purposes only and does not constitute medical advice. Always consult a qualified healthcare professional.

## 🚀 Getting Started

Herbora requires **no build step and no dependencies**. Just clone and open.

### Run locally

```bash
git clone https://github.com/gabrielbogdan5/Herbora-web.git
cd Herbora-web
```

Then open `index.html` directly in your browser, or serve it with any static server:

```bash
# Using Node.js
npx serve .

# Using Python
python -m http.server 8000
```

### Deploy (free options)

| Platform | How |
|---|---|
| **GitHub Pages** | Enable in repo Settings → Pages → Deploy from branch `main` |
| **Netlify** | Drag & drop the folder at [netlify.com/drop](https://netlify.com/drop) |
| **Vercel** | `npx vercel` in the project folder |

## 📱 PWA Installation

Herbora can be installed as a native-feeling app:

- **Android:** Open in Chrome → tap the menu → *Add to Home Screen*
- **iOS:** Open in Safari → tap Share → *Add to Home Screen*

For full PWA support, add icon files to an `icons/` folder:
```
icons/
├── icon-192.png   # 192×192 px
└── icon-512.png   # 512×512 px
```

## 🔒 Privacy

Herbora is built with a **privacy-first** approach:

- ✅ No user accounts or registration
- ✅ No analytics or tracking scripts
- ✅ No data sent to any server
- ✅ All preferences stored locally on your device (localStorage)
- ✅ No ads

The only external request is loading fonts from [Google Fonts](https://fonts.google.com).

See the full [Privacy Policy](./privacy-policy.html).
